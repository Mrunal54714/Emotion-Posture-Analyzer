# Emotion & Body Posture Analyzer

## Overview
This project is a Computer Vision application that performs real-time analysis of human emotions and physical posture. It provides a downloadable summary of the analysis for user tracking.

## Features
- **Emotion Detection:** Identifies facial expressions in real-time.
- **Posture Analysis:** Monitors body alignment and positioning.
- **Data Export:** Allows users to download session reports.

## Tech Stack
- **Language:** Python
- **Libraries:** streamlit, opencv, mediapipe, deepface, numpy, pandas, matplotlib, Pillow, tensorflow, tf-keras
